# 工具

## 1.查看消耗资源最多的线程

```shell
top -H -p $pid
$pid: 要分析程序的进程ID
```

我们将用top命令得出pid，[换算](http://tool.oschina.net/hexconvert)成十六进制， 例如：0x34ac8。

拿到了ID如何去对应堆栈信息里面的ID呢？

## 2. 使用jstack命令打出堆栈信息

```shell
jstack $pid
$pid: 要分析程序的进程ID
```

执行上述命令后，会打印出pid的堆栈信息，为了方便起见，可以重定向到文件中，然后使用grep命令查找字符串0x34ac8。

## 3.CPU耗时分析

```shell
bash busy_threads.sh -p $pid -c $cnt
$pid: 要分析程序的进程ID
$cnt: top排名个数
```
